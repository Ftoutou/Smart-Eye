# Smart Eye  
### Multi-Modal Medical Imaging & Reporting Platform

Smart Eye is a **cross-modality medical imaging platform** built for ophthalmologists, radiologists, and clinicians.  
It supports **DICOM, NIfTI, OCT, fundus, MRI, CT, ultrasound**, and standard image formats â€” all in a single, powerful desktop interface.

Built using **Python, PyQt5, and modern medical imaging libraries**, Smart Eye integrates visualization, reporting, and tele-consultation into one smooth workflow.

---

## Features

### Patient Management  
- Create, search, filter, and organize patients  
- Automatic folder structure  
- Timeline view of all exams, reports, and images  
- Right-click actions: open, rename, delete, export  

---

### Multi-Modal Medical Viewer  
Supports:  
- **DICOM**
- **NIfTI**
- **JPEG / PNG / BMP**
- **OCT / Fundus / Angiography**

Tools included:  
- Zoom, pan, rotate  
- Annotations (drawing, erasing)  
- 2D/3D navigation  
- Slice scrolling  
- Image cropping & export  
- Planned: full 3D reconstruction & volumetrics  

---

### Advanced Report Editor  
- Rich-text medical report editor  
- Insert floating images  
- Resize, drag, zoom images interactively  
- Smart PDF export with exact positions  
- Automatically saved inside patient folder  

---

### Tele-Consultation (Microsoft Teams API)  
- Built-in **Open Teams integration**  
- Start a remote consultation directly from the patient view  
- Share screens and images instantly  

---

## Technologies Used

### Core  
- Python  
- PyQt5  
- NumPy / SciPy  
- OpenCV  

### Medical Imaging  
- NiBabel (NIfTI)  
- PyDicom (DICOM)  
- SimpleITK  
- VTK / PyVista  
- napari  

### Reporting & Graphics  
- QPainter  
- Qt PDF Export  

### Integration  
- Microsoft Teams API  
- Modular multi-window architecture
